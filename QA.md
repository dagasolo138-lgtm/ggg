# 验收

每次发布后在手机浏览器检查：

- 首次打开没有弹窗。
- 新建、重命名、收藏、删除对话可用。
- 刷新后对话和设置仍在。
- 发送、停止、重试可用。
- 文本附件可添加和移除。
- Artifact 可预览、编辑、保存、下载。
- 抽屉、菜单、弹窗可关闭，输入框不被遮挡。

## 真实对话验收

在站点 URL 后加入 `?qa=1`。此入口不会出现在正常界面。

1. 先在正常页面填写临时、限额的 API Key。
2. 使用带 `?qa=1` 的地址打开验收面板。
3. 手动点击“运行验收”。它会发出 7 次真实请求，结果不会写入聊天、记忆或设置。
4. 检查“待人工检查”项目，复制或下载 JSON 后再关闭面板。
5. 测试完成后清除临时 Key，或在 DeepSeek 控制台撤销。
